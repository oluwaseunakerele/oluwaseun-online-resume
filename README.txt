# Oluwaseun Online Resume

This folder contains a single page online resume and PDFs.

## Quick Publish (GitHub Pages)
1. Create a new public GitHub repo named `oluwaseun-online-resume`.
2. Upload all files from this folder (index.html and PDFs) to the repo root.
3. In the repo, go to Settings → Pages → Build and deployment → Branch: `main` (root).
4. Wait for it to publish. Your link will be:
   https://USERNAME.github.io/oluwaseun-online-resume/

Replace USERNAME with your GitHub username.

## Quick Publish (Netlify)
- Drag and drop this folder at https://app.netlify.com/drop
- Netlify gives you a live URL immediately.
